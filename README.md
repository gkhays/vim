# vim
VIM configuration files

```viml
colorscheme wombat

set shiftwidth=4
set tabstop=4
set smarttab
set autoindent
set smartindent
set nowrapscan
set lines=40
set columns=120
set ruler
set scrolloff=5

syntax on

" folding settings

set foldmethod=syntax
set foldnestmax=20
set foldlevel=0
set nofoldenable

set number
```
